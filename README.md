# Reglet SDKs

Official Software Development Kits (SDKs) for building plugins for the [Reglet](https://github.com/reglet-dev/reglet) compliance platform.

## Available SDKs

*   **[Go SDK](go/)** (`reglet-sdk-go`): Native Go support for writing efficient, type-safe WASM plugins.
